Developer

Do you want to be a developer? Well, all you need to do is help postLit by coding a feature, fixing a bug, or anything else. Read more...

The developer badge is a badge that shows that you have contributed to postLit, the wiki, or anything else related to the site. The badge shows up on the user's profile and the explore page, like all other badges.

Additional Information

Icon

The official icon displayed on a developer's profiles and the explore page.

Requirements

The only requirment to get this badge is to contribute to the code of the wiki, the actual site, or in any other way. Then, you can ask rgantzos for the badge! It's really that simple :)
